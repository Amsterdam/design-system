# Breekpunt

We hanteren 2 breekpunten waar het ontwerp van een website zich aanpast voor een goede gebruikerservaring: bij schermbreedtes van 576px en 1088px.

In _css_ werken we _mobile first_: eerst maken we het ontwerp onder het breekpunt en vervolgens gebruiken we een _media query_ om het ontwerp boven het breekpunt te maken.

Bijvoorbeeld:

```css
.class {
  /* Your narrow styling */

  @media screen and (min-width: $amsterdam-breakpoint-wide) {
    /* Your wide styling */
  }
}
```

Binnen deze repository kun je het breakpoint als Sass-variabele importeren. Je vindt deze variabele in [packages/css/utils/breakpoint.scss](https://github.com/Amsterdam/design-system/blob/main/packages/css/utils/breakpoint.scss).
