### Zonder opsommingsteken

Gebruik dit component niet alleen voor tekstuele opsommingen, maar ook voor lijsten van grotere secties zoals een overzicht van artikelen of een verzameling zoekresultaten. Semantisch zijn dit immers ook lijsten.

Gebruikers van schermlezers krijgen op deze manier ook informatie over de lengte van de lijst en het volgnummer van het item waarop ze focus zetten.

Je maakt zo’n ‘onzichtbare’ lijst via de `no-marker` modifier.
