# Breekpunt

We hanteren 1 breekpunt waar het ontwerp van een website zich aanpast voor een goede gebruikerservaring op verschillend formaat apparaten: 854px breed.

In _css_ werken we _mobile first_: eerst maken we het ontwerp onder het breekpunt en vervolgens gebruiken we een _media query_ om het ontwerp boven het breekpunt te maken.
