# Focus indicator

De toegankelijkheidsrichtlijnen eisen dat het element op de pagina dat focus heeft visueel herkenbaar is. Dit geldt ook voor formuliervelden. Houd rekening bij het ontwerp en de ontwikkeling van de website of applicatie met de focus indicator.

Wij gebruiken grotendeels de standaard focus indicator van de webbrowsers. De enige aanpassing die we doen is de focus `outline` een `offset` geven van 2 pixels, zodat er ruimte zit tussen het element en de focus indicator. Hierdoor heeft de focus indicator ook voldoende contrast met donkere elementen.

## Relevante WCAG eisen

- [WCAG eis 2.4.7](https://www.w3.org/TR/WCAG22/#focus-visible): de focus indicator is zichtbaar
- [WCAG eis 2.4.11](https://www.w3.org/TR/WCAG22/#focus-not-obscured-minimum): de focus indicator wordt niet volledig verborgen door content
- [WCAG eis 2.4.12](https://www.w3.org/TR/WCAG22/#focus-not-obscured-enhanced): de focus indicator wordt niet deels verborgen door content
- [WCAG eis 2.4.13](https://www.w3.org/TR/WCAG22/#focus-appearance): de focus indicator is minimaal 2 pixels breed en heeft voldoende contrast met het element dat focus heeft
