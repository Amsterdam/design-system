# Links

Several link components share the same appearance in various aspects.
This set of common tokens maintains their visual consistency.

## Default

Our default links have the following styles:

<DesignTokensTable exclude={["subtle", "contrast", "inverse"]} path="ams.links" showDescriptions tokens={tokens} />

## Subtle

Subtle links are used in places where the context helps indicate that the text is a link.
For instance, in the Page Header or Pagination components.
As a result, their styling can be less obvious.

<DesignTokensTable path="ams.links.subtle" tokens={tokens} />

## Contrast

Links on light (but not white) backgrounds use the interactive contrast colour.

<DesignTokensTable path="ams.links.contrast" tokens={tokens} />

## Inverse

On dark backgrounds, links have the inverse colour.

<DesignTokensTable path="ams.links.inverse" tokens={tokens} />

## How to use

Use these tokens if you recreate our React components with different technology, or if you create a new link component.
