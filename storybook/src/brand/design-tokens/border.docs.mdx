# Border

Elements that have a border, outline or underline use one of these widths.

## Design

We use a limited set of border widths for consistency.

## Tokens

### Spacious Mode

The default border is 2 pixels thick.
One thinner and two thicker options are available.

| CSS variable                 |     Value | Value (px) | Example                                             |
| :--------------------------- | --------: | :--------: | :-------------------------------------------------- |
| `var(--ams-border-width-s)`  | 0.0625rem |     1      | <BorderSample width="var(--ams-border-width-s)" />  |
| `var(--ams-border-width-m)`  |  0.125rem |     2      | <BorderSample width="var(--ams-border-width-m)" />  |
| `var(--ams-border-width-l)`  | 0.1875rem |     3      | <BorderSample width="var(--ams-border-width-l)" />  |
| `var(--ams-border-width-xl)` |   0.25rem |     4      | <BorderSample width="var(--ams-border-width-xl)" /> |

### Compact Mode

All borders are a pixel thinner in [Compact Mode](/docs/docs-developer-guide-modes--docs), except for the small width.

| CSS variable                 |     Value | Value (px) | Example                                             |
| :--------------------------- | --------: | :--------: | :-------------------------------------------------- |
| `var(--ams-border-width-s)`  | 0.0625rem |     1      | <BorderSample width="var(--ams-border-width-s)" />  |
| `var(--ams-border-width-m)`  | 0.0625rem |     1      | <BorderSample width="var(--ams-border-width-m)" />  |
| `var(--ams-border-width-l)`  |  0.125rem |     2      | <BorderSample width="var(--ams-border-width-l)" />  |
| `var(--ams-border-width-xl)` | 0.1875rem |     3      | <BorderSample width="var(--ams-border-width-xl)" /> |
