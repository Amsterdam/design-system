# Typography

We have a fluid typography system.
The options for the font family, font size, line height and font weight come in two groups: body text and headings.

## Body text

### Font size

The size of text adjusts to the width of the window for pleasant reading on screens of any size.

### Line height

The line heights have been designed to match the corresponding font size.
Do not combine tokens of different sizes.

| CSS variable                                          | Value |
| :---------------------------------------------------- | :---: |
| `var(--ams-typography-body-text-small-line-height)`   |  1.6  |
| `var(--ams-typography-body-text-line-height)`         |  1.8  |
| `var(--ams-typography-body-text-large-line-height)`   |  1.6  |
| `var(--ams-typography-body-text-x-large-line-height)` |  1.4  |

### Font weight

The font weight can either be regular or bold.

| CSS variable                                       | Value | Example                                                                              |
| :------------------------------------------------- | :---: | :----------------------------------------------------------------------------------- |
| `var(--ams-typography-body-text-font-weight)`      |  400  | <span class="ams-docs-token-example--font-weight-regular">Default font weight</span> |
| `var(--ams-typography-body-text-bold-font-weight)` |  800  | <span class="ams-docs-token-example--font-weight-bold">Bold font weight</span>       |

## Other aspects

### Font family

All text appears in our Amsterdam Sans font.
If it is not available, we use Arial or the generic sans-serif instead.

| CSS variable                        |                Value                |
| :---------------------------------- | :---------------------------------: |
| `var(--ams-typography-font-family)` | 'Amsterdam Sans', Arial, sans-serif |

### Hyphenation

Words that do not fit on a single line will be split with a hyphen and continue on the next line.
This happens automatically, but only for words of 12 or more characters.
There must be at least 8 characters before the hyphen and 4 characters after it.
This rule helps keep the text readable and prevents strange hyphenations.

| CSS variable                                  | Value  |
| :-------------------------------------------- | :----- |
| `var(--ams-typography-hyphenate-limit-chars)` | 12 8 4 |

### Underlining

We only underline [hyperlinks](/docs/brand-design-tokens-links--docs).

### Letter spacing

Kerning has been optimized in the font file.
We do not adjust the space between glyphs.

## Compact mode

Our large font sizes for websites would be counterproductive in applications.
They use the compact theme, where both body text and headings are considerably smaller.
