# Space

We have a single scale of spacing values for all kinds of whitespace.

## Spacious

Medium space is as large as the font size of regular body text.
Two smaller and three larger sizes are available.

Because our typography is fluid, spacing is as well.
For example, the medium space is 16 pixels at a window width of 320, and grows to 24 pixels at a window width of 1600.
The larger space levels grow considerably slower on narrow windows.

| CSS variable           | Example                                                                              | ≤&nbsp;320 | 832  | ≥&nbsp;1600 | Example                                                                                          |

| `var(--ams-space-sm)`  | <div className="ams-docs-token-example--space" style={{ inlineSize: '0.75rem' }} />  |     12     |  12  |     12      | <div className="ams-docs-token-example--space" style={{ inlineSize: 'var(--ams-space-sm)' }} />  |
| `var(--ams-space-md)`  | <div className="ams-docs-token-example--space" style={{ inlineSize: '1rem' }} />     |     16     | 19.2 |     24      | <div className="ams-docs-token-example--space" style={{ inlineSize: 'var(--ams-space-md)' }} />  |

## Compact

In Compact Mode, spacing is not responsive.
The medium space is 16 pixels, also matching the regular font size.

| CSS variable           |     | Example                                                                                                |
| :--------------------- | --: | :----------------------------------------------------------------------------------------------------- |
| `var(--ams-space-xs)`  |   4 | <div className="ams-theme--compact ams-docs-token-example--space" style={{ inlineSize: '0.25rem' }} /> |
| `var(--ams-space-sm)`  |   8 | <div className="ams-theme--compact ams-docs-token-example--space" style={{ inlineSize: '0.5rem' }} />  |
| `var(--ams-space-md)`  |  16 | <div className="ams-theme--compact ams-docs-token-example--space" style={{ inlineSize: '1rem' }} />    |
| `var(--ams-space-lg)`  |  24 | <div className="ams-theme--compact ams-docs-token-example--space" style={{ inlineSize: '1.5rem' }} />  |
| `var(--ams-space-xl)`  |  32 | <div className="ams-theme--compact ams-docs-token-example--space" style={{ inlineSize: '2rem' }} />    |
| `var(--ams-space-2xl)` |  40 | <div className="ams-theme--compact ams-docs-token-example--space" style={{ inlineSize: '2.5rem' }} />  |

## About the units we use

We use pixels in our documentation on sizes to convey a more accurate mental image.
In code, sizes are defined in relative units (like `rem`, `em`, `ex`, `vw`, `vh`, and `ch`) to make components more flexible and accessible.
MDN offers a good overview on these [CSS values and units](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Values_and_Units).
