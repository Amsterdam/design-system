# Links

Several input components share the same styles.
This set of common tokens shows that connection.

For now, only one shared token exists.

| CSS variable                    | Value | Description                                                                                                                                 |
| :------------------------------ | :---- | :------------------------------------------------------------------------------------------------------------------------------------------ |
| `var(--ams-inputs-line-height)` | `1.4` | Inputs and Text Area have a smaller line height than body text. Buttons also use this, to match the height of Inputs and align them easily. |
