# Article page

This page type is for news or similar kinds of articles.

## Headings

- Use a level 1 Heading for the title of the article.
- Use level 2 Headings for the section titles.

## Spacing

- Add a small margin bottom to a Heading 2 with following body text.
- Use an extra large margin between sections.
