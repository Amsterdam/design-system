# Handbook page

Use this template for long-form reference content that is split across many short pages, such as a personnel handbook or a policy document.

A collapsible [Table of Contents](/docs/components-navigation-table-of-contents--docs) on the left lists all pages, with the current page and its ancestors pre-expanded and `aria-current="page"` on the active link.
The page content on the right uses the [Prose](/docs/utilities-css-prose--docs) utility class so headings, lead paragraphs, and body copy are spaced without hand-picked margins.
