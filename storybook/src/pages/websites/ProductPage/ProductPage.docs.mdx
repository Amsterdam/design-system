# Product page

Lists all information about a product or service of the City.
Often includes a Call to Action Link that directs the user to a form.

## Headings

- Use a level 1 Heading for the title of the page.
- Use level 2 Headings for the section titles.

## Spacing

- Add a small margin bottom to a Heading 2 with following body text.
- Use an extra large margin between sections and before the Call to Action Link.
