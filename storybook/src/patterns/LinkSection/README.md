# Link Section

A small block that provides one or more links, with a Heading and an optional Paragraph.

## Anatomy

A link section consists of the following elements:

1. [Heading](https://designsystem.amsterdam/?path=/docs/components-text-heading--docs) with `level={2 | 3 | 4}` and `size="level-3"`
2. [Paragraph](/docs/components-text-paragraph--docs) – an optional introduction
3. Either a [Link List](/docs/components-navigation-link--docs) – for a list of links,
4. Or a [Standalone Link](https://designsystem.amsterdam/?path=/docs/components-navigation-standalone-link--docs) – for a single link

## Layout

- The parent Grid Cell follows the [sizing guidelines for Cells](/docs/pages-public-introduction--docs#how-to-size-the-grid-cells).

## Spacing

- The Heading has a bottom margin of `s` if it is level 2, `xs` for level 3, `0` for level 4.
- Between the Paragraph and the Link List or Standalone Link, the spacing is `m`.
