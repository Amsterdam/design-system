### Native scrollbar

Turning on `nativeScrollbar` will make the slider use the native scrollbar. You might need this if you disable to other controls.

### Source set images

To use source set images, you need to pass an array of objects with `srcSet` and `src` properties. The `srcSet` property should be a string with the image URLs separated by commas. The `src` property should be the URL of the image to use as a fallback.
