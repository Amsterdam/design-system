### Sizes

It may be visually more suitable to display a heading in a smaller font size.
Use the `size` prop for this (but sparingly).
Ensure that you still select a level that accurately represents the section’s hierarchy.
These, for example, are all Headings with level 1:

<Canvas of={HeadingStories.Sizes} />

### Inverse colour

Use the inverse colour if the Heading sits on a dark background.
This ensures the colour of the text provides enough contrast.

<Canvas of={HeadingStories.InverseColour} />

### With an icon

Headings with level 3 to 6 can be paired with an icon.
Use a [Row](/docs/components-layout-row--docs) and the `size` prop of [Icon](/docs/components-media-icon--docs) to align them.
