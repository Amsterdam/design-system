## How to use

Use the React Icon component together with a React SVG component from `@amsterdam/design-system-react-icons`.
Import this SVG as follows:

```tsx
import { EmailIcon } from "@amsterdam/design-system-react-icons";
```

Then, you can use it in the component like this:

```tsx
<Icon svg={EmailIcon} />
```
