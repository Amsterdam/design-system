## Examples

### With side menu

Add `layout="application"` if your page uses a [Menu](/docs/components-navigation-menu--docs).
Add the Menu as a direct child, next to the [Page Header](/docs/components-containers-page-header--docs), main content and [Page Footer](/docs/components-containers-page-footer--docs), and give it the `inWideWindow` prop.
Include a copy of the Menu, without that prop, in the Page Header.
