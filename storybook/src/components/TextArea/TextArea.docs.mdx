### In a Field

Usually, you should place a Text Area inside a Field component.
Here’s an example of a Field component that includes a Label, a description, and a Text Area.
If you don’t need the description, just take out the description Paragraph and the `aria-describedby` from the Text Input.

For more information, check [the Field component documentation](/docs/components-forms-field--docs).
