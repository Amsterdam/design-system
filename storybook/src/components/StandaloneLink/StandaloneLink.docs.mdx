## Usage guidelines

### When to use

Use a Standalone Link for a single link on its own line of text.

### When not to use

Within running body text, use a regular [Link](/docs/components-navigation-link--docs) instead.

For additional guidelines, refer to the [Link](/docs/components-navigation-link--docs) component.

Use [Margin](/docs/utilities-css-margin--docs) or [Prose](/docs/utilities-css-prose--docs) to add vertical whitespace.

## Examples

### With heading and paragraph

A Standalone Link often appears in a block with a [Heading](/docs/components-text-heading--docs) and an introducing [Paragraph](/docs/components-text-paragraph--docs) to lead to a follow-up page.
Use a size of ‘level-3’ for the Heading, add a small bottom margin, and a medium bottom margin to the Paragraph.

<Canvas of={StandaloneLinkStories.WithHeadingAndParagraph} />

### Custom icon

If it helps understand the link, a different icon than the default chevron can be used.

<Canvas of={StandaloneLinkStories.WithCustomIcon} />

### On a coloured background

A Standalone Link on a coloured background must set [the correct text colour](/docs/brand-design-tokens-colour--docs#pairing-foreground-with-background-colours) to provide enough contrast.
We have lighter and darker background colours, and links behave differently on each.

#### Contrast colour

Use the contrast colour on a lime, orange or yellow background.
It will make the Standalone Link black.

<Canvas of={StandaloneLinkStories.ContrastColour} />

#### Inverse colour

Use the inverse colour on an azure, blue, green, magenta or purple background.
It will make the Standalone Link white.

<Canvas of={StandaloneLinkStories.InverseColour} />

## Design

The Standalone Link has a chevron icon at the start to signify that it is a link.
This also makes it visually identical to a single link in a [Link List](/docs/components-navigation-link-list--docs).
As a result, pages can have sections with either one or several links used interchangeably; they look alike, and their semantics are correct.
