## Examples

### Multiple descriptions

A term may have multiple descriptions.

<Canvas of={DescriptionListStories.MultipleDescriptions} />

### Multiple terms

In the case of multiple consequtive terms for a description you need to
wrap all `DescriptionList.Term` and `DescriptionList.Description` pairs in `DescriptionList.Group`.
And it will also require the minWidth property to be set to anything but `auto` to prevent the misalignment of different groups.

<Canvas of={DescriptionListStories.MultipleTerms} />

### Rich description

A description can include rich content such as inline formatting, links, paragraphs, and even lists.

<Canvas of={DescriptionListStories.RichDescription} />

### Inverse colour

Set the `inverseColor` prop if the Description List sits on a dark background.
This ensures the colour of the text provides enough contrast.

<Canvas of={DescriptionListStories.InverseColour} />
