## Examples

### With heading

A list of links often appears in a block with a [Heading](/docs/components-text-heading--docs).
Use a size of ‘level-3’ for the Heading and add a small bottom margin.

<Canvas of={LinkListStories.WithHeading} />

### Custom icons

Replace the list icon with a more meaningful one to help understand a link more quickly or easily.
Avoid using both semantic and list icons in the same list.

<Canvas of={LinkListStories.CustomIcons} />

### Small text

We use a smaller font size for text in a Figure Caption and for metadata like dates or categories.
When using a list in small text, set the prop for each item in the list.

<Canvas of={LinkListStories.SmallText} />

### Link

This example of a single link allows using the controls below.
The Link List Link renders both the `<li>` and `<a>` elements.
Any attributes and the `ref` are passed on to the `<a>` element.

<Canvas of={LinkListStories.Link} />

<Controls of={LinkListStories.Link} />

### On a coloured background

A Link on a coloured background must set [the correct text colour](?path=/docs/brand-design-tokens-colour--docs#pairing-foreground-with-background-colours) to provide enough contrast.
We have lighter and darker background colours, and links behave differently on each.

#### Contrast colour

Use the contrast colour on a lime, orange or yellow background.
It will make the Link black.

<Canvas of={LinkListStories.ContrastColour} />

#### Inverse colour

Use the inverse colour on an azure, blue, green, magenta or purple background.
It will make the Link white.

<Canvas of={LinkListStories.InverseColour} />
