## Usage guidelines

### When to use

Quote someone’s literal written or oral text.

Give a Blockquote a width of half or a whole column of text.
Use [Margin](/docs/utilities-css-margin--docs) or [Prose](/docs/utilities-css-prose--docs) to add vertical whitespace around it.

### When not to use

### How to use

If the citation is in a language other than that of the surrounding content, indicate so through the `lang` attribute.

## Examples

### Inverse colour

Use the inverse colour if the Blockquote sits on a dark background.
This ensures the colour of the text provides enough contrast.
