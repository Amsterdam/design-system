## Keyboard support

The user can navigate the links with their keyboard:

| key         | element                                        |
| :---------- | :--------------------------------------------- |
| Tab         | Go to the next element that can have focus     |
| Shift + Tab | Go to the previous element that can have focus |
| Enter       | Follow the link that has focus                 |
