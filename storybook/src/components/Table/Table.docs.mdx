## Subcomponents

### Cell

Holds a single value within a row.
Use the `align` prop to align the content to the center or end of the cell.
Apply the same value to all cells in the column and the matching `Table.HeaderCell`.

<Canvas of={TableCellStories.Cell} />

<Controls of={TableCellStories.Cell} />

## Examples

### With footer and row headers

Use table headers to tell users what the rows and columns represent.
Use the scope attribute to help users of assistive technology distinguish between row and column headers.
Use `Table.Footer` for a summary row.

<Canvas of={TableStories.WithFooterAndRowHeaders} />

### With heading and description

When the table needs context – a source, a date, or reading instructions – place a heading and a small paragraph above it and hide the caption visually.

<Canvas of={TableStories.WithHeadingAndDescription} />

### With caption in figure

The context can be positioned below the table as well.
In this case, use a [Figure](/docs/components-media-figure--docs) and let the Table keep its Caption.

<Canvas of={TableStories.InFigure} />
