## Subcomponents

A Table is composed of a Caption, a Header containing Header Cells, a Body, an optional Footer, and Rows of Cells.
Caption, Header, Body, Footer, and Row have no props of their own; the example above shows each of them in place.

### Cell

Holds a single value within a row.
Use the `align` prop to align the content to the center or end of the cell.
Apply the same value to all cells in the column and the matching `Table.HeaderCell`.

### Header Cell

Labels the column of values below it.
Use the `align` prop to match the alignment of the column’s cells.

## Usage guidelines

### How to use

Use [Margin](/docs/utilities-css-margin--docs) or [Prose](/docs/utilities-css-prose--docs) to add vertical whitespace.

Add a caption so users can find and understand the Table without reading its data.
Use a [Heading](/docs/components-text-heading--docs) in it, at the level that fits the document outline.

To place content between the Heading and the Table, keep the Caption but hide it visually, then position the heading and content above the Table.
Do not use ARIA relations; they hurt the Table's semantics (see [this article](https://www.scottohara.me/blog/2019/04/19/tabled-tables.html)).

## Examples

### With footer and row headers

Use table headers to tell users what the rows and columns represent.
Use the scope attribute to help users of assistive technology distinguish between row and column headers.
Use `Table.Footer` for a summary row.

<Canvas of={TableStories.WithFooterAndRowHeaders} />

### With heading and description

When the table needs context – a source, a date, or reading instructions – place a heading and a small paragraph above it and hide the caption visually.

<Canvas of={TableStories.WithHeadingAndDescription} />

### With caption in figure

The context can be positioned below the table as well.
In this case, use a [Figure](/docs/components-media-figure--docs) and let the Table keep its Caption.

<Canvas of={TableStories.InFigure} />

## Accessibility

A Table caption creates the accessible name for the Table, announced by screen readers on every encounter.
