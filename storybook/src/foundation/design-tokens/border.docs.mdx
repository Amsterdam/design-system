# Border

Elements that have a border, outline or underline use one of these widths.

## Widths

We have 4 border widths:

| Size            | px  |  rem   | Token name            | Example                                                                                   |
| :-------------- | :-: | :----: | :-------------------- | ----------------------------------------------------------------------------------------- |
| **Small**       |  1  | 0.0625 | `ams.border.width.sm` | <div style={{ borderInlineStart: "var(--ams-border-width-sm) solid", height: '1rem' }} /> |
| **Medium**      |  2  | 0.125  | `ams.border.width.md` | <div style={{ borderInlineStart: "var(--ams-border-width-md) solid", height: '1rem' }} /> |
| **Large**       |  3  | 0.1875 | `ams.border.width.lg` | <div style={{ borderInlineStart: "var(--ams-border-width-lg) solid", height: '1rem' }} /> |
| **Extra large** |  4  |  0.25  | `ams.border.width.xl` | <div style={{ borderInlineStart: "var(--ams-border-width-xl) solid", height: '1rem' }} /> |
