# Headings

Several components that use headings share the same styles.
This set of common tokens shows that connection.

## Levels

The levels have the following styles:

| CSS variable                            | Value                           |
| :-------------------------------------- | :------------------------------ |
| `var(--ams-headings-level-0-font-size)` | `var(--ams-font-size-4x-large)` |
| `var(--ams-headings-level-1-font-size)` | `var(--ams-font-size-3x-large)` |
| `var(--ams-headings-level-2-font-size)` | `var(--ams-font-size-2x-large)` |
| `var(--ams-headings-level-3-font-size)` | `var(--ams-font-size-x-large)`  |
| `var(--ams-headings-level-4-font-size)` | `var(--ams-font-size-large)`    |
| `var(--ams-headings-level-5-font-size)` | `var(--ams-font-size)`          |
| `var(--ams-headings-level-6-font-size)` | `var(--ams-font-size-small)`    |
