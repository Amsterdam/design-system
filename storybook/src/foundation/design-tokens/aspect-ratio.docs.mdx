# Aspect Ratio

Constrain elements like images and videos to one of the available aspect ratios.

## Widths

We have 6 named aspect ratios:

| Token name                 | ratio  | Example                                                                                                                         |
| :------------------------- | :----: | ------------------------------------------------------------------------------------------------------------------------------- |
