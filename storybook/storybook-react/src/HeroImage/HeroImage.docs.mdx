## Voorbeelden

### Basis

De beeldverhouding van de afbeelding is 32:9.
Dat komt overeen met de `2x-wide` [aspect ratio](?path=/docs/react_layout-aspect-ratio--docs).

Kies een voldoende grote afbeelding.
Als het nodig is wordt deze uitgerekt om de volledige oppervlakte in te nemen.

### Met zoekveld

Zo kan bijvoorbeeld een zoekveld geplaatst worden.
