## Stories

### Composed Accordion

Een accordion bestaat uit meerdere accordionsecties. Gebruik minimaal 3 en maximaal 10 secties onder elkaar. Je kunt door een accordion navigeren met je toetsenbord:

| key                  | element                                                                                                                      |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Enter of Space       | Open of sluit de accordionsectie die de focus heeft                                                                          |
| Tab                  | Ga naar het volgende element dat focus kan hebben                                                                            |
| Shift + Tab          | Ga naar het volgende element dat focus kan hebben                                                                            |
| Pijltje naar beneden | Als een accordionsectie focus heeft: ga naar de volgende sectie. Als de laatste sectie focus heeft, ga naar de eerste sectie |
| Pijltje omhoog       | Als een accordionsectie focus heeft: ga naar de vorige sectie. Als de eerste sectie focus heeft, ga naar de laatste sectie   |
| Home                 | Als een accordionsectie focus heeft: ga naar de eerste sectie                                                                |
| End                  | Als een accordionsectie focus heeft: ga naar de laatste sectie                                                               |

### Expanded by default

Als je wilt dat de inhoud van een accordionsectie al getoond wordt als de gebruiker op de pagina komt, geef dan de `expanded` _property_ mee.

## Too many landmarks

Een accordionsectie gebruikt _default_ de HTML tag _section_ om de inhoud in te renderen.
Als je veel accordionsecties op je pagina hebt (meer dan 6), dan kan dit leiden tot een teveel aan _landmark_ regio's, wat verwarrend kan zijn voor gebruikers van schermlezers.
Gebruik in dat geval `section={false}` voor je accordionsectie.

## Technische toelichting

Op het moment (28-6-2023) heeft het HTML element `details` wat toegankelijkheidsproblemen.
[Zo wordt een _heading_ in een `summary` niet goed begrepen door schermlezers](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/summary#summaries_as_headings) en
leest NVDA de ingeklapte en en uitgeklapte status niet goed voor in Chrome en Edge. Daarom hebben we er voor gekozen om het _native_ element niet te gebruiken.
Als deze problemen zijn opgelost, zouden we dit wel kunnen doen.
