## Stories

### Expanded by default

Als je wilt dat de inhoud van een accordionsectie al getoond wordt als de gebruiker op de pagina komt, geef dan de `expanded` _property_ mee aan het `Accordion.Section` component.

<Canvas of={AccordionStories.ExpandedByDefault} />

## Too many landmarks

Een accordionsectie gebruikt standaard de HTML tag `section` om de inhoud in te renderen.
Als je veel accordionsecties op je pagina hebt (meer dan 6), dan kan dit leiden tot een teveel aan _landmark_ regio's, wat verwarrend kan zijn voor gebruikers van schermlezers.
Gebruik in dat geval `section={false}`. Let op: Storybook geeft deze _property_ niet goed weer in het codeblock.

## Technische toelichting

Op het moment (28-6-2023) heeft het HTML element `details` wat toegankelijkheidsproblemen.
[Zo wordt een _heading_ in een `summary` niet goed begrepen door schermlezers](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/summary#summaries_as_headings) en
leest NVDA de ingeklapte en en uitgeklapte status niet goed voor in Chrome en Edge. Daarom hebben we er voor gekozen om dit _native_ element niet te gebruiken.
Als deze problemen zijn opgelost, zouden we dit wel kunnen doen.
