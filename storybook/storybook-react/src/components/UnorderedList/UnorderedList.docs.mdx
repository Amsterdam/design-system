# Unordered list

Een lijst is een verticale groep gerelateerde inhoud.

## Voorbeelden

### Basis

Een ongeordende lijst heeft vierkante opsommingstekens.
Tussen de items staat extra witruimte.
Dat geeft een beter onderscheid tussen de items, zeker als ze uit meerdere regels tekst bestaan.

<Canvas of={UnorderedListStories.Default} />

### Twee niveaus

Een lijst mag één dieper niveau hebben.
In dit geval wordt een kastlijntjes (‘en dash’) gebruikt als opsommingsteken.
Dat is links uitgelijnd met de tekst van het bovenliggende niveau.

<Canvas of={UnorderedListStories.TwoLevels} />

### Lijst zonder opsommingstekens

Een overzicht van gelijkwaardige elementen is ook een ongeordende lijst.
Gebruik in dit soort gevallen een lijst zonder opsommingstekens om die semantiek aan te brengen.
Daardoor kunnen schermlezers helpen de lijst te begrijpen en te navigeren.
