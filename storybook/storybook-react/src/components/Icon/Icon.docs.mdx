# Gebruik

Het React Icon component gebruik je samen met een React SVG component uit `@amsterdam/design-system-react-icons`. Importeer deze SVG als volgt:

```js
import { Email } from "@amsterdam/design-system-react-icons";
```

Daarna kun je deze in het component gebruiken op deze manier:

```js
<Icon svg={Email} />
```
