# Ordered list

Een geordende lijst is een verticale groep gerelateerde inhoud met een volgorde.

## Voorbeelden

### Basis

De lijst heeft oplopende getallen als opsommingstekens.
Er is genoeg ruimte voor getallen tot 99.
Tussen de items staat extra witruimte.
Dat geeft een beter onderscheid tussen de items, zeker als ze uit meerdere regels tekst bestaan.

<Canvas of={OrderedListStories.Basis} />

### Twee niveaus

Een lijst mag één dieper niveau hebben.
In dit geval worden onderkast letters gebruikt als opsommingsteken.

<Canvas of={OrderedListStories.TweeNiveaus} />

### Hoger startgetal

Wanneer nodig kan de opsomming beginnen met een hoger getal.
Gebruik het HTML-attribuut `start`.

### Omgekeerde volgorde

Ook kunnen de cijfers achterstevoren lopen.
Gebruik het HTML-attribuut `reversed`.

### Lijst zonder opsommingstekens

Een overzicht van artikelen of zoekresultaten is ook een geordende lijst.
Gebruik in dit soort gevallen een lijst zonder opsommingstekens om die semantiek aan te brengen.
Daardoor kunnen schermlezers helpen de lijst te begrijpen en te navigeren.

In dit geval verwacht de lijst dat componenten in de items de typografie definiëren.
