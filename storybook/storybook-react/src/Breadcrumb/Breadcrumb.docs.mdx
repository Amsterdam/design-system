## Gebruik

- Het `<Breadcrumb />` accepteert een lijst van `<Breadcrumb.Item />` componenten als kinderen. Andere types worden er uitgefilterd en dus niet getoond.
- Het `<Breadcrumb.Item />` accepteert een `href` prop, die de link bepaalt. Deze is verplicht om mee te geven.
- Het `<Breadcrumb.Item />` gaat maximaal tot 8 levels diep. Items die dieper zijn worden niet getoond.
