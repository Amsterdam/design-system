De roze vlakken vertegenwoordigen de kolommen van het grid. Zoom in of verander de breedte van het venster om de breedtes en het aantal kolommen te zien veranderen.

Er is een `GridCell` component waarop kan worden ingesteld welke kolommen van het grid het element beslaat.

De cellen worden automatisch op de eerstvolgende beschikbare ruimte in het grid geplaatst.

<Controls />

### Drie kolommen

Content kan in drie kolommen worden weergegeven door via `gridColumns={4}` elk element vier kolommen van het grid te laten beslaan.

<Canvas of={PageGridStories.ThreeColumns} />

### Een blok in het midden

Dit blok beslaat de tien middelste kolommen, van nummer 2 tot en met 11. In dit geval moet dus expliciet worden aangegeven bij welk kolomnummer het element begint: `gridColumns={{ start: 2, span: 10 }}`.

### Paragrafen rechtstreeks op het grid

Paragrafen accepteren ook de `gridColumns` property van het `GridCell` component en kunnen dus rechtstreeks op het grid worden geplaatst. Het is de bedoeling dit door te voeren voor alle componenten.
