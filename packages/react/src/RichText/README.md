# React RichText component

[Rich text documentation](../../../css/src/components/rich-text/README.md)

Use `RichText` when editors need to enter formatted content that is stored as HTML, but must be limited to the elements provided by the Amsterdam Design System.

The component renders a small formatting toolbar and a `contenteditable` area that behaves like a multiline text field.
