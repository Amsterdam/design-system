# Skeleton

A loading placeholder for async content. See Storybook for usage and props.

## Guidelines

- Use Skeleton to indicate that content is loading and will appear soon.
- Place Skeletons where users expect content, such as in cards, lists, or form fields.
- Prefer the rectangle variant for blocks of content, circle for avatars, and text for paragraphs or lines.
- Use animation to signal activity; disable animation for static placeholders.
- Ensure sufficient contrast and accessibility for all variants.
