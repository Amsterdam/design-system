# Description List

A collection of terms and their descriptions.

## Design

- The terms and descriptions stack vertically by default. They sit next to each other when their [[query container](https://claude.ai/docs/utilities-css-query-container--docs)](/docs/utilities-css-query-container--docs) is wide enough (at lest `32rem`).
- The column for the terms is as wide as the longest term, without wrapping. Its width can be adjusted to be ‘large’ (50%), ‘medium’ (33%), or ‘small’ (20%), which also allows the terms to wrap.
- The term is set in bold text.
